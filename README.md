# LLM Chatbot (Python)

This is a simple chatbot built using OpenAI API.

## Features
- Takes user input
- Generates AI response
- Runs in terminal

## Tech Used
- Python
- OpenAI API

## How to Run
1. Install openai
2. Add your API key
3. Run the script

## Note
Do not share your API key publicly.
